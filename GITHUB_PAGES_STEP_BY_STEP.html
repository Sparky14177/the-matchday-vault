Host “The Matchday Vault” on GitHub Pages — Mobile Guide

  1. Sign in at github.com
  2. Create repo: the-matchday-vault (Public)
  3. Open repo → Add file → Upload files
  4. From Files app, select all contents from the unzipped folder and upload
  5. Go to Settings → Pages → Source = main / /(root) → Save
  6. Visit: https://Sparky14177.github.io/the-matchday-vault/ — then “Add to Home Screen”
